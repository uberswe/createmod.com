{{ T .Language "Are Minecraft schematics safe?" }}

{{ T .Language "Short answer: schematics are data, not programs. Here is exactly what that means, and what we check anyway." }}

{{ T .Language "Schematics cannot run code on your computer" }}

{{ T .Language "A schematic file (.nbt, .schem, .litematic) is a structured list of blocks and their positions, the same kind of data as a save file. Unlike a mod .jar, it contains no executable code, and Minecraft never executes it. Opening or downloading a schematic cannot install anything on your machine. A file pretending to be a schematic could still be malformed on purpose, which is why we validate structure before anything else." }}

{{ T .Language "What a schematic CAN contain" }}

{{ T .Language "Once pasted into a world, some blocks act on the world or players:" }}

  • {{ T .Language "Command blocks" }}: {{ T .Language "run any command when powered, including /give or /op. Only a risk in worlds where command blocks are enabled." }}
  • {{ T .Language "Mob spawners" }}: {{ T .Language "spawn entities continuously." }}
  • {{ T .Language "Signs and books with click commands" }}: {{ T .Language "run a command when a player clicks them." }}
  • {{ T .Language "Structure and jigsaw blocks" }}: {{ T .Language "can load or generate additional structures." }}

{{ T .Language "None of these are malicious by themselves; command blocks power plenty of legitimate contraptions. But you should know they are there before you paste." }}

{{ T .Language "What CreateMod.com checks" }}

{{ T .Language "1. File hardening (every upload)." }} {{ T .Language "Size caps, decompression-bomb guards, nesting-depth and length validation. Malformed or hostile files are rejected at upload." }}

{{ T .Language "2. Content inspection (every published schematic)." }} {{ T .Language "We parse every block and block entity and record a transparency manifest: command blocks (with their command text), spawners (with what they spawn), sign and book click commands, structure blocks. The manifest is shown on the schematic page." }}

{{ T .Language "When the inspection finds nothing notable, the schematic shows a Validated badge. When it finds something, the badge says what: \"contains 2 command blocks\", with the details one click away. We deliberately avoid a blanket \"100% secure\" stamp and only make claims we actually checked." }}
