API Documentation

Welcome to the createmod.com API. This page provides a quick overview and examples. More endpoints will be added over time.

Authentication & API keys

Authentication uses API keys via the HTTP Authorization header.

Authorization: Bearer <API_KEY>

How to generate API keys

  1. Go to Settings when logged in.
  2. Find the "API Keys" section and click "Generate new key".
  3. Copy the key and store it securely. You won't be able to view it again.

Note: key revocation and usage tracking per key will be rolled out gradually.

Endpoints

  • Search schematics: GET /api/schematics?query=<term>&page=1
  • Schematic detail: GET /api/schematics/{name} — returns images, size, blocks, materials list (no full NBT)

Examples

Search for schematics with the term "windmill":

curl -H "Authorization: Bearer <API_KEY>" \
  "https://createmod.com/api/schematics?query=windmill&page=1"

Fetch a schematic detail by name:

curl -H "Authorization: Bearer <API_KEY>" \
  "https://createmod.com/api/schematics/<name>"

Rate limits

To keep the service reliable, requests may be rate-limited per API key. Limits and usage tracking will be documented here.

Code samples

Go example:

req, _ := http.NewRequest("GET", "https://createmod.com/api/schematics?query=gear", nil)
req.Header.Set("Authorization", "Bearer <API_KEY>")
resp, _ := http.DefaultClient.Do(req)
